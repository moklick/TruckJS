Truck Anim

Truck Anim is a JavaScript module for making simple and elegant animations with CSS3 effects.

Examples

$.set(prop, val)

Set prop to val.

          
$.anim('#example-1 .box')
.set('margin-left', '200px')
.run();

$.add(prop, val)

Increment prop by val, where val is a Number (click several times).

          
$.anim('#example-2 .box')
.add('margin-left', 200)
.run();

$.sub(prop, val)

Decrement prop by val, where val is a Number (click several times).

          
$.anim('#example-3 .box')
.sub('margin-left', 100)
.run();

$.rotate(deg)

Rotate by deg.

          
$.anim('#example-4 .box')
.rotate(140)
.run();

$.duration(n)

Set animation duration to n which is a Number or a string such as "4s".

          
$.anim('#example-5 .box')
.set('background-color', 'blue')
.duration('2s')
.run();

$.translate(x[, y])

Translate x and optionally y axis. Aliased as anim.to(x[, y]).

          
$.anim('#example-6 .box')
.translate(300, 80)
.run();

$.x(n) / anim.y(n)

Translate x or y axis. Aliased by anim.translateX(n) and anim.translateY(n).

          
$.anim('#example-7 .box')
.x(300)
.y(20)
.run();

$.skew(x[, y])

Skew x, and optionally y. anim.skewX(n) and anim.skewY(n) are also available.

          
$.anim('#example-8 .box')
.x(300)
.skew(50)
.set('height', 20)
.run();

$.scale(x[, y])

Scale the x, and optionally y axis. anim.scaleX(n) and anim.scaleY(n) are also available.

          
$.anim('#example-9 .box')
.scale(3)
.run();

$.ease(fn)

Use the given easing fn.

default
in
out
in-out
snap
(0,1,1,0)
          
$.anim('#example-10 .box1').x(400).run();
$.anim('#example-10 .box2').ease('in').x(400).run();
$.anim('#example-10 .box3').ease('out').x(400).run();
$.anim('#example-10 .box4').ease('in-out').x(400).run();
$.anim('#example-10 .box5').ease('snap').x(400).run();
$.anim('#example-10 .box6').ease('cubic-bezier(0,1,1,0)').x(400).run();

setTimeout(function(){
  $.anim('#example-10 .box1').x(0).run();
  $.anim('#example-10 .box2').x(0).run();
  $.anim('#example-10 .box3').x(0).run();
  $.anim('#example-10 .box4').x(0).run();
  $.anim('#example-10 .box5').x(0).run();
  $.anim('#example-10 .box6').x(0).run();
}, 1200);

$.run([fn])

The end() method triggers the animation to play, optionally invoking the callback fn when complete.

          
$.anim('#example-11 .box')
.set('background-color', 'red')
.duration(1000)
.run(function(){
  $.anim('#example-11 .box')
  .set('background-color', 'white')
  .run();
});

$.delay(n)

Set animation delay to n which is a Number or a string such as "4s".

          
$.anim('#example-12 .box')
.set('background-color', 'blue')
.delay('2s')
.run();

$.then([fn])

Defer an action such as invoking a fn, end()ing the given Anim instance, or returning a clone for chaining. The anim.pop() method is used to return the current Anim instance's parent in the chain.

          
var moveBack = $.anim('#example-13 .box')
.set('background-color', 'white')
.x(0);

$.anim('#example-13 .box')
.set('background-color', 'red')
.x(500)
.then(moveBack)
.run();

$.anim('#example-13 .box2')
.set('background-color', 'red')
.x(500)
.scale(.5)
.rotate(60)
.then()
  .rotate(30)
  .scale(1.5)
  .set('border-radius', 5)
  .set('background-color', 'white')
.then()
  .set('opacity', 0)
.pop()
.pop()
.run();

$.select(selector)

This function is used throughout anim to select elements. For example if we wanted to utilize jQuery, we could re-define this function as shown below.

          
$.anim.select = function(selector){
  return $(selector).get(0);
};

$.defaults

Defaults used throughout Anim, simply re-define to apply a new default.

          
$.anim.defaults = {
  duration: 500
};
